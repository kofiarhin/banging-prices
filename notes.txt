## 1. Git Workflow (Origin + Heroku)


* Rebase from both remotes first

## 2. Canonical Product Data Shape (Normalized)

```json
{
  "_id": "ObjectId",
  "canonicalKey": "store:hash",
  "store": "boohoo",
  "storeName": "Boohoo",
  "title": "Black Oversized Hoodie",
  "category": "hoodies",
  "gender": "women",

  "price": 12.99,
  "originalPrice": 29.99,
  "discountPercent": 57,
  "currency": "GBP",

  "image": "https://...jpg",
  "images": ["https://...jpg"],

  "productUrl": "https://www....",
  "saleUrl": "https://www....",

  "inStock": true,
  "status": "active",

  "lastSeenAt": "2026-01-14T10:00:00.000Z",
  "createdAt": "2026-01-14T10:00:00.000Z",
  "updatedAt": "2026-01-14T16:00:00.000Z"
}
```

**Key insight**

* `canonicalKey = store + product fingerprint` → dedupe across crawls
* `lastSeenAt` ≠ `updatedAt` (important for availability logic)
